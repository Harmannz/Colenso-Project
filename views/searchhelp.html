How to search!

Following describes how to perform text, markup and nested searches

Text Search

  1. Enclose text to search within "quotation"
    • "William"
  2. Use logical operators such as:
    • "William" AND "Colenso"
    • "William" OR "Colenso"
    • "William" && "Colenso"
    • "William" || "Colenso"
    • NOT and ! are also supported

    3. You can use any combination of logical operators

  3. Text search can also perform complicated seraching such as:
    • "William" ft:or "Colenso"
    • "William" ft:and "Colenso"
    • As well as operations specified here

Markup Search

  1. Type XPATH queries and click markup search button such as:
    • //author
    • //name[@type='place']
  2. Markup search caters for full XPATH queries as appropriate

Nested Search

Nested search allows you to perform search on documents returned by previous serach by clicking on the nested search radio button when searching

Simply check the nested search button and select the appropriate search type to perform nested search.

With nested search you have the opportunity to instantly download all the documents returned by the nested search.

Simply click the Download All? button

Happy Searching!
